# M.T ポートフォリオサイト

## ■ 概要
未経験からWeb運用・ECサイト改善を目指すM.Tのポートフォリオ。
実務想定で制作した3作品を通して、更新業務・改善・UI設計のスキルをアピール。

## ■ 収録作品
1. **EC運用想定：商品登録・更新管理ツール（GAS）**
   - 自動集計＋トリガー通知で更新効率向上

2. **Shopify想定：商品LP改善ページ**
   - 購買導線を意識した改善設計

3. **CMS更新デモ（簡易CMS風UI切替）**
   - ボタン切替でコンテンツ変更が可能、非エンジニア向けUI

## ■ 使用技術
- HTML / CSS / JavaScript / GAS / Spreadsheet
- DOM操作 / UI切替 / レスポンシブ対応

## ■ 特徴
- 更新業務を想定した実務志向
- 非エンジニアでも扱える設計
- 課題発見→改善→確認のフローを意識
